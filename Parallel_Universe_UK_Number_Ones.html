About this dataset (pipeline + caveats) 
Parallel-Universe UK Number Ones (2016 – May 2026)
What this is
Every weekly UK Official Singles Chart #1 from mid-May 2016 through 16 May 2026 (deduplicated by song),
fed through Suno AI as a Simple-mode description prompt. For each real song, the corresponding row of the
"Tracks" sheet shows the prompt used and links to the two parallel-universe Suno tracks Suno generated from it.
Pipeline
1. Researched the 133 unique UK #1s from year-by-year Wikipedia chart pages.
2. Wrote a 300-550 char musical description for each (genre, tempo, instrumentation, vocal style, lyrical theme).
3. Submitted each description to Suno (Simple mode, Pro plan) via browser automation.
4. Captured every Suno-returned clip ID via a fetch interceptor and built song URLs.
5. Re-submitted any that did not return clip IDs on the first attempt until all 133 mapped.

Caveats
- Some 2025-2026 entries are recent enough that chart-position certainty is lower; cross-check with Wikipedia if needed.
- "Last Christmas" (Wham!) is listed once even though it hit #1 in multiple Decembers.
- Suno auto-titles each generated track; the title in your workspace will not match the original song title.
- Earlier diagnostic submissions left some duplicate tracks in your Suno workspace beyond the 266 final pairs linked here.
